# Puzzle Challenge – Solutions

This repository contains my solutions for the 3-problem challenge.

---

##  Problem 1 – Grid Rotation (Clue 1)
- Read a grid from input
- Rotate each row left or right based on the given directions
- Extract the middle row
- Convert characters to ASCII and sum them
- This ASCII sum becomes *Clue 1*

---

##  Problem 2 – String Processing (Clue 2)
Steps I followed:
1. Reverse the string  
2. Remove every 3rd character  
3. Shift each character's ASCII by +2  
4. Count vowels  
This vowel count becomes *Clue 2*

---

##  Problem 3 – State Machine (Clue 3)
- Take numbers from input  
- Move them through a series of state rules  
- Count how many reach the DONE state  
- That count becomes *Clue 3*
